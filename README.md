## Overview

Luminar Neo Professional Suite is a complete AI-enhanced photography platform that covers the spectrum of editing tasks from technical RAW corrections through dramatic creative transformations in a single non-destructive editing environment.

Extension marketplace allows additional AI tools and creative packs to be installed, expanding the capability set beyond the core application with specialized tools for specific photography styles and subjects.

Plugin integration with Lightroom and Photoshop embeds Luminar Neo's AI tools within existing professional workflows, allowing photographers to apply Luminar effects without migrating their entire workflow.

Layer-based compositing supports complex multi-image compositions where AI-powered processing is applied to individual layers, enabling sophisticated creative works that combine AI processing with traditional layer editing.

Luminar Neo Professional Suite is the photo editing platform for photographers who embrace AI-assisted editing and want a tool where artificial intelligence genuinely does the heavy lifting on the most time-consuming editing tasks.

## Performance & Integration

Luminar Neo is engineered around macOS technologies that handle performance-critical work efficiently. Tasks run in the background without slowing down the rest of your system — you can keep working while Luminar Neo processes operations quietly behind the scenes.

The app integrates deeply with the macOS ecosystem out of the box:

- Launch workflows directly through **Spotlight**
- Preview files instantly with **Quick Look**
- Trigger actions from any app via the **Services menu**
- Fully compatible with **Shortcuts** for automated workflows
